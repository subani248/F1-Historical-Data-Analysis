# 🏎️ Formula1 Race Data Analysis & Power BI Dashboard

A comprehensive data analytics project that explores historical Formula 1 race data using **Python** for data cleaning and exploratory data analysis (EDA), and **Power BI** for interactive dashboard creation and business intelligence.

---

## 📌 Project Overview

This project analyzes historical Formula 1 race results to uncover trends, driver performance, and championship insights through data visualization and statistical analysis.

The project consists of two parts:

- **Python Analysis**
  - Data Cleaning
  - Exploratory Data Analysis (EDA)
  - Feature Engineering
  - Driver Performance Analysis
  - Statistical Analysis
  - Data Visualization

- **Power BI Dashboard**
  - Interactive Dashboard
  - KPI Cards
  - Trend Analysis
  - Driver Performance Dashboard
  - Race Statistics Dashboard
  - Dynamic Filters and Slicers

---

## 📂 Dataset

**Dataset:** `AllRace.csv`

The dataset contains historical Formula 1 race information including:

- Race Name
- Race Date
- Driver finishing positions (1st–35th)

## 📈 Key Findings

- Formula 1 has experienced significant growth in the number of races over the decades.
- A small group of elite drivers has dominated race victories throughout the championship's history.
- Drivers with more podium finishes generally record higher win counts.
- Consistent drivers maintain lower average finishing positions across seasons.
- Most Formula 1 races are held between **May and October**.
- Modern Formula 1 seasons feature significantly more races than earlier decades.
